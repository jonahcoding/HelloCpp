命令解释器：
    shell(unix系统)
    bash(Linux系统)

常用命令：
    data        查看时间
    history     显示历史命令
    clear       清屏（ctrl+l）

快捷键
    ctrl+p      向上滚动历史命令
    ctrl+n      向下滚动历史命令   
    ctrl+b      光标向前移动
    ctrl+f      光标向后移动
    ctrl+a      光标移至行首
    ctrl+e      光标移至行尾
    ctrl+h      删除光标前的字符
    ctrl+d      删除光标后的字符（包括光标）
    ctrl+u      删除光标前的内容

目录结构（树）
    /: 根目录
        /bin    存放常用命令文件（可执行程序）
        /boot   系统启动文件
        /dev    外部设备文件
        /etc    配置文件
        /lib    动态链接共享库
        /home   用户主目录
        /lost+found     非法关机产生的文件
        /media  挂载媒体设备
        /mnt    临时挂载文件系统（如光驱），进入查看光驱中文件     
        /opt    第三方软件安装目录
        /proc   内存映射（虚拟目录）
        /root   超级用户目录
        /sbin   管理员命令文件目录
        /tmp    存放临时文件
        /usr    存放应用程序与文件

用户目录
    cd -        切换目录

软链接（文件/目录）
    ln -s 原文件绝对路径 快捷方式
硬链接（仅文件）
    ln 原文件名 新文件名 

文件相关操作
    查找
        find 路径 -name "文件名"
        find 路径 -size +10k    /   find 路径 -size +10M
    检索
        grep -r "查找的内容" 查找路径

软件安装
    查询是否已安装
            rpm -qa xxx
    在线安装
            sudo yum install 软件包名
    deb包安装
            安装    dpkg -i XXX.deb
            卸载    dpkg -r XXX
    源码安装
            （先查看README说明）
            解压缩源码包，cd进入
            创建Makefile，检测编译环境./configure
            make编译源码
            安装 sudo make install
            卸载 sudo make distclean

ftp服务器搭建（文件上传与下载）
    vsftpd
        安装    sudo yum install vsftpd
        修改配置文件    sudo vim /etc/vsftpd.conf 
                        # 允许匿名用户登录
                        anonymous_enable=YES
                        # 实名用户拥有写权限（上传）
                        write_enable=YES
                        # 本地掩码022
                        local_umask=022
                        # 匿名用户可上传数据
                        anon_upload_enable=YES
                        # 匿名用户可创建目录
                        anon_mkdir_write_enable=YES
        重启服务    service vsftpd restart

        客户端实名登录      ftp 服务器IP
        上传文件（非目录）  put 文件名（登录ftp前的目录下文件）
        下载文件（非目录）  get 文件名
        退出               bye

        匿名登录           ftp 服务器IP    /   用户名：anonymous，  密码：回车跳过
        限定匿名用户只能访问指定目录
                    修改ftp配置文件：
                        # 指定匿名用户根目录，默认目录为 /srv/ftp/
                         anon_root=/home/jonah/anonymousFTP/
        为使匿名用户有操作文件需要修改权限。

        lftp工具（可对文件夹操作）
            安装    sudo yum install lftp
            实名登录    lftp username@ip
            匿名登录    lftp 服务器IP login
            切换目录    lcd 本机目录
            上传文件/多个文件    put/mput 
            下载/下载文件       get/mget
            下载目录            mirror
            上传目录            mirror -R

nfs服务搭建（挂载文件共享目录）
    安装    sudo yum -y install nfs-utils rpcbind
    修改配置文件    sudo vim exports
                   # /home/jonah/NFS 192.168.111.0/24(rw,no_root_squash,no_all_squash,sync)
                   /home/jonah/NFS *(rw,no_root_squash,no_all_squash,sync)
    配置生效    sudo exportfs -r
    重启服务    service rpcbind start

    客户端挂载  mount -t nfs ip:被挂载的目录 挂载位置
